Edições de texto

Tag small para diminuir o texto.

Tag MARK para marcação de texto. A cor é padrão mas pode ser alterada com STYLE ou CSS.

Itálico e negrito.

A tag <b> destaca uma palavra mas não dá sentido (não é semântica)

A tag <strong> dá o sentido de importância ou destaque.

A tag <i> deixa em itálico mas não dá sentido (não semântica)

A tag EM dá o sentido de ÊNFASE a palavra ou frase (tag semântica, que dá sentido)

Para endereços, ao invés de usar a tag EM, usamos a tag
