Áudios dentro do site

Utilizamos a tag <audio>

Caso seu navegador não consiga reproduzir o áudio, faça o download aqui

Atributo controls vai apresentar o player em tela.

  • metadata: vai carregar apenas as informações sobre o arquivo (tempo, direitos, tamanho, etc)
  • none: não vai carregar nenhuma informação a respeito do arquivo
  • auto (padrão): vai carregar o arquivo de áudio inteiro assim que a página for carregada (não recomendado)

Dentro da tag audio, podemos adicionar vários source para formatos diferente do mesmo áudio, para caso o navegador não reproduza um dos formatos, possa tentar o outro.
