Criação de listas

A tag OL (Ordered Lists ou Listas Ordenadas)

  1. Item 1
  2. item 2
  3. Item 3

As listas podem usar o parâmetro type para criar listas com ordenações diferentes

Podem ser usadas os valores de type

  • 1 - Padrão, como o exemplo anterior, cria uma lista ordenada por números sequênciais
  • A - Ordena de forma alfabética maíuscula
  • a - Ordena de forma alfabética com letras minúsculas
  • I - Ordena com números romanos maíusculos
  • i - Ordena com números romanos minúsculos

Exemplos:

  1. Item 1
  2. item 2
  3. Item 3
  1. Item 1
  2. item 2
  3. Item 3
  1. Item 1
  2. item 2
  3. Item 3

Listas não ordenadas (unordered lists) utilizam a tag UL

A tag UL também faz uso do parâmetro TYPE. A diferença é que este utiliza os valores:

disc - Padrão. Uma bola preta preenchida

  • Item 1
  • item 2
  • Item 3

circle - Uma bola com borda

  • Item 1
  • item 2
  • Item 3

square - Um pequeno quadrado preenchido

  • Item 1
  • item 2
  • Item 3
