Código-fonte

O comando document.getElementById('teste') é usado na linguagem JavaScript. Para o nome do comando, podemos usar a tag CODE

Quando quiser escrever um código no site e quiser exibir ele com a indentação, a tag CODE deverá estar dentro da tag PRE.

Tag CODE sem a tag PRE

SELECT * FROM PES_PESSOA WHERE PES_ID >= 200

Tag CODE dentro da tag PRE

        
            SELECT *
                FROM PES_PESSOA
            WHERE PES_ID >= 200

Citações

Para marcar uma citação, o ideal é utilizar a tag <q> que irá colocar as aspas e dar sentido a citação

Como disse um sábio: Mais vale um pássaro no ar, do que 2 no cú.

Abreviações

Podemos usar a tag ABBR para que as siglas/abreviações sejam explicadas. Como por exemplo, o jogo WoW, ou HTML

Colocar textos ou letras "baixas", como por exemplo H20, utilizamos a tag sub. E para deixar o texto "sobre" as palavras, por exemplo em matemática, usamos a tag sup.

sub: H2O

sup: X10
